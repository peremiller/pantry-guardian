# Security policy

Please do not disclose vulnerabilities in a public issue. Contact the repository owner privately through their GitHub profile with a minimal reproduction, affected paths, and suggested severity.

Never submit an OpenAI key or other credential. If a secret is exposed, revoke it immediately, create a replacement, update the hosting secret, and review usage logs.

Pantry Guardian does not guarantee that food is safe. Expiry dates, storage history, package condition, allergens, and visible or odor-based spoilage must be assessed by the user.
